Flexbox

Flexbox gồm 2 thành phần chính: flex container và flex item. Flex container là thẻ cha, bên trong sẽ có một hoặc nhiều thẻ con, được gọi là flex item.

Lưu ý: flex item phải là thẻ con trực tiếp của thẻ cha (flex container) bạn nhé.

flex-grow

Định lại kích thước width của các item trong flexbox Property Value:
  • flex-grow: 0 - giá trị mặc định => width sẽ bằng chiều dài nội dung của item đó
  • flex-grow: 1 - number => vùng chứa các flex-item sẽ tràn ra sao cho phủ đầy width của flex-container
  • flex-grow: 2 - number => vùng chứa sẽ to gấp 2 lần so với các vùng chứa khác.
Example 1: tuỳ chỉnh vùng chứa text 2 với width gấp 3 lần các vùng chứa khác
  • .item:nth-child(1), .item:nth-child(3) { flex-grow: 1;}
  • .item:nth-child(2) { flex-grow: 3;}

flex-shrink

Kiểm soát sự co lại của các item khi có nhiều flex item trong flex container Property Value:
  • flex-shrink: 0 - numbers => vùng chứa sẽ không bị Co lại, giữ nguyên độ rộng ban đầu của flex-item
  • flex-shrink: 1 - giá trị mặc định => vùng chứa sẽ bị Co lại, đều như nhau
  • flex-shrink: 2 - numbers => vùng chứa sẽ bị Co lại, co gấp 2 lần so với các vùng khác
  • flex-shrink: 3 - number => vùng chứa sẽ bị Co lại, co gấp 3 lần so với các vùng khác
Example: ban đầu ta cho các item có width: 20%; giá trị mặc định flex-shrink; 1

flex-basis

Thuộc tính chỉ định độ dài ban đầu của flex item Property Value:
  • auto: mặc định: -> width = chính chiều dài chứa nó.
  • number: đơn vị độ dài hoặc % -> chỉ định độ dài ban đầu của flex item
Example:
  • .item { flex-basis: 30%} -> khai báo width ban đầu cho flex-item
  • .item { flex-shrink: 0;} -> vùng chứa sẽ không bị co lại khi có thêm nhiều flex-item
  • .item { flex-grow: 1;} -> vùng chứa các flex-item sẽ tràn ra sao cho phủ đầy width của flex-container
  • .item:nth-child(2) { flex-basis: 100px} -> tuỳ chỉnh vùng chứa text 2 với width=100px
